== Description ==
com status with the latest post that you've made.  It also has the option of including a link back to your post so can people can easily click on it to view the post.

This plugin was updated from Victoria's original plugin to include the use of TinyURL because URLs can be very long and Twitter has a maximum character amount of 140 characters.  This will make your URLs shorter for posting.

== Installation ==


1. Upload `twitter-updater_v2.zip` to the `/wp-content/plugins/` directory
2. activate the plugin through the `Plugins` menu in WordPress
3. Go to Manage -> Twitter Updater, select the options you want to use, then verify your Twitter login details
4. That's it!  You're all set and ready to update your Twitter status with post updates
